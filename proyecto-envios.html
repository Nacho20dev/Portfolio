Volver Ignacio Egozcue | Case Study

Agencia de Envíos

Sistema de gestión logística empresarial desarrollado con Clean Architecture, priorizando el desacoplamiento y la integridad de datos.

.NET Core 8 C# / MVC SQL Server Entity Framework JWT & Security

Ingeniería en Capas

El núcleo del sistema implementa una arquitectura de capas donde las reglas de negocio son el centro. Este diseño permite que el software sea agnóstico a la base de datos o al cliente que lo consume.

Backoffice Funcionarios

Gestión administrativa de ingresos, despachos y estados operativos de paquetes.

Endpoints de Clientes

Interfaz API para consulta de tarifas y tracking de envíos en tiempo real.

Estructura de la Solución
Presentación (WebAPI / MVC / Controllers)
Aplicación (Casos de Uso / Interfaces)
Dominio (Entidades / Reglas de Negocio)
Infraestructura (EF Core / Seguridad / SQL)
Seguridad

Autenticación basada en JWT y algoritmos de Hasheo para protección de credenciales.

Persistencia

Uso de Entity Framework Code-First para una gestión de datos robusta y escalable.

Inyección

Implementación de Dependency Injection para cumplir con los principios SOLID.

Infraestructura Cloud-Ready

Diseñado para entornos de alta disponibilidad. El sistema fue validado y desplegado exitosamente en Microsoft Azure (App Services & SQL Database).
