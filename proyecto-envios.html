Agencia de Envíos Enterprise

Solución logística robusta basada en Clean Architecture. Implementación de lógica de negocio compleja con persistencia en SQL Server mediante LINQ to Entities.

.NET 8 LINQ to Entities JWT Authentication Entity Framework

Arquitectura de Datos & Consultas

El sistema utiliza Entity Framework Core con un enfoque Code-First. La manipulación de datos se realiza exclusivamente mediante LINQ (Language Integrated Query), garantizando consultas fuertemente tipadas y optimizadas para SQL Server.
