Engineering Case Study

LOGIVIP

Ecosistema escalable para la gestión logística satelital. Una solución integral desarrollada bajo estándares de ingeniería de software, modelado UML y arquitectura robusta en .NET Core 8.

ASP.NET MVC SQL SERVER ENTITY FRAMEWORK

Documentación de Análisis

El proyecto está sustentado por una memoria técnica de **140 páginas** que detalla el ciclo de vida de desarrollo. No es solo código; es una arquitectura planificada.

  • +50 Casos de Uso detallados.
  • Especificación de Requerimientos Funcionales y No Funcionales.
  • Planificación de seguridad y matriz de roles.

Modelado UML & Arquitectura

Diseño estructural mediante **Diagrama de Clases y Diagramas de Secuencia** para garantizar la integridad de los datos en el rastreo de envíos.

Módulos del Sistema

Gestión Interna

Panel para funcionarios con auditoría de cargas, gestión de estados y filtrado avanzado por parámetros técnicos.

Rastreo en Tiempo Real

Integración de lógica satelital para el seguimiento de paquetes mediante ID de Tracking único por cliente.

Persistencia de Datos

Base de datos SQL Server normalizada con Entity Framework Core para una manipulación eficiente de la información.

Código de Alta Fidelidad

Implementación de controladores dinámicos y vistas Razor optimizadas. El sistema utiliza **ViewModels** para desacoplar la capa de datos de la presentación, asegurando un código limpio y mantenible.
