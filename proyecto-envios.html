Especificaciones Técnicas (RF)

RF1 & RF2
Acceso y Auditoría

Login de empleados por roles (Admin/Funcionario). Sistema de auditoría Code-First que registra Quién, Cuándo y Qué (Acción) en cada alta o baja de staff.

RF3 & RF5
Flujo de Envío

Ingreso dinámico de envíos (Email, Agencia Destino, Peso). Gestión de estados etapa a etapa mediante comentarios de seguimiento vinculados al tracking.

RF4 & RF6
Control y Entrega

Listado de envíos en proceso con cierre de ciclo mediante fecha de entrega real. Consulta profunda de detalles por número de seguimiento único.

Capa de Datos Avanzada

Para manejar la complejidad de múltiples sucursales, implementé consultas optimizadas con LINQ to Entities, permitiendo un filtrado eficiente directamente en el motor de base de datos SQL Server.

// Ejemplo de consulta de seguimiento nacional
var tracking = await _context.Envios
.Include(e => e.SucursalDestino)
.Where(e => e.TrackingId == inputId)
.ToListAsync();
