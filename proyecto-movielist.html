EL DESAFÍO TÉCNICO

MovieList nació como un proyecto para dominar la gestión de datos en aplicaciones de gran escala. Implementé un sistema de catálogo donde el usuario puede explorar, filtrar y gestionar películas mediante una API REST.

El foco principal fue la **Arquitectura de Estado Global**. Utilicé Redux para centralizar la información de las películas, los filtros y el estado de carga, asegurando que la interfaz reaccione de forma instantánea a cada interacción.

STACK DE DESARROLLO

  • Store Central: Manejo de estado con Redux.
  • API Integration: Consumo de datos externos (JSON/REST).
  • ES6 Essentials: Uso intensivo de Map, Filter y Asincronismo.
  • Postman: Testing exhaustivo de la estructura de la API.
