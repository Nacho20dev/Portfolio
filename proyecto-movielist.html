Catálogo Dinámico de Películas con Estado Global

React Vanilla
Redux (Store)
HTML5
ES6

Descripción y Objetivos

MovieList fue mi primer gran desafío en el mundo de React, donde pasé de hacer componentes simples a manejar una aplicación estática pero con lógica compleja. El objetivo fue crear una interfaz que listara películas y permitiera gestionarlas, poniendo el foco en la gestión eficiente del estado de la aplicación.

Aunque el frontend es estático, la lógica de negocio se maneja íntegramente en el cliente, lo que me obligó a entender cómo estructurar datos y flujos de información sin depender de un backend en tiempo real.

Skills Técnicos

  • Implementación de **React.js Vanilla**
  • Uso Profundo de **Redux** para el Store
  • Manejo de **Reducers, Actions y State**
  • Estructura de datos en ES6 (Map, Filter)
  • Diseño responsive con HTML5 y CSS

Desafíos Superados

  • Entender el flujo de datos unidireccional de Redux.
  • Evitar la mutabilidad del estado de forma manual.
  • Sincronizar el renderizado de React con el Store de Redux.
  • Optimizar los componentes para que no re-rendericen innecesariamente.
