MovieList

React Redux JS Vanilla HTML5

Un sistema de gestión cinematográfica que combina la semántica pura de HTML, la lógica algorítmica de JavaScript y la eficiencia de Redux para el manejo de datos globales.

Ingeniería del Proyecto

La base del proyecto reside en JavaScript Vanilla, utilizado para manipular arrays de datos y realizar filtrados complejos por fecha y categoría.

Lógica Pura

Uso de métodos funcionales de JS para procesar la información de la API.

Estado Global

Redux Toolkit actúa como la única fuente de verdad para el listado de películas.
