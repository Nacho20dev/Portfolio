Sobre el Desarrollo

Goalify fue diseñado para resolver la falta de conexión visual entre personas que comparten objetivos de bienestar similares. El mayor desafío técnico fue optimizar el consumo de la API de Develotion para que la carga de datos geográficos en el mapa fuera instantánea, garantizando una fluidez total en dispositivos móviles.
