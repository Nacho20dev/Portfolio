Sobre el Desarrollo

Goalify fue diseñado para resolver la falta de conexión visual entre personas que comparten objetivos de fitness similares. Durante el desarrollo, el mayor desafío fue optimizar el consumo de la API de Develotion para que la carga de datos en el mapa fuera instantánea y fluida.
